## Introduction

The fieldbus section of the config defines the structure of the fieldbus and is usually not to be altered.

## Tips for defining Fieldbus configs

The fieldbus section of the config defines the offsets of the objects transferred over the fieldbus interface from &gbc to the receiving process.

If you need to modify this then it is best to contact us first.

## Example JSON

    "0": {
      "jointCount": 10,
      "RxPdo": {

}
