## Introduction

The fieldbus section of the config defines the structure of the fieldbus and is usually not to be altered.

## Details

`"jointCount"`

RxPdo object

RxPdo?: FieldbusRxPdoLayout
TxPdo object

see schema

TxPdo?: FieldbusTxPdoLayout
Number of joints stored in the fieldbus process data

jointCount?: number
