## Introduction

The tool section of the config defines the "table" of tools used by the machine.

Tools have diameters and a translation and rotation.

## Details

`"diameter"`- optional

`"translation"` - vector 3

`"rotation"` quaternion

## Example JSON

```json
"tool": {
        "default": {
        },
        "offsetz": {
            "translation": {
                "z": -10
            }
        }
    }
```
