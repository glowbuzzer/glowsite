# Gcode in GBC

<Tldr>
    - GBC does interpret gcode - this is done in the toolkit's React components
    - Supports xx
    - xxx
</Tldr>

Whilst GBC doesn't actually interpret the gcode itself (this is done in the React component) this section discusses how to use gcode with GBC when using the toolkit's gcode React components.

The glowbuzzer toolkit supports most of the standard gcodes with a handfful of variations, omissions and extensions that are described here.

There should be no issues exporting gcode from standard CAM tools if care is taken with the post-processor.
