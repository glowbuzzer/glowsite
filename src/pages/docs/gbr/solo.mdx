# {props.subtitle}

The solo activity API allows you to execute individual activities and wait for them to complete (using promises).
If you are not concerned with blending moves, this provides a convenient way to sequence moves together from your
&fe code. The example below shows a basic rounded rectangle made up of individual moves.

```jsx
const api=useSoloActivity(0)
await api.moveLine(0, 10).promise()
await api.moveArc(2,12).centre(2,10).direction(ARCDIRECTION.ARCDIRECTION_CW).promise()
await api.moveLine(10, 12).promise()
await api.moveArc(12,10).centre(10,10).direction(ARCDIRECTION.ARCDIRECTION_CW).promise()
await api.moveLine(12, 0).promise()
await api.moveArc(10,-2).centre(10,0).direction(ARCDIRECTION.ARCDIRECTION_CW).promise()
await api.moveLine(2, -2).promise()
await api.moveArc(0,0).centre(2,0).direction(ARCDIRECTION.ARCDIRECTION_CW).promise()
```

If blending is important you may want to use the [G-Code hook](gcode).
