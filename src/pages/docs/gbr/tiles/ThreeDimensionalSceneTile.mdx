## Customising the default tile

The following example shows how to customise the standard 3D scene tile shown above.

```jsx demo tile toolpath wide
import React from "react"
import {DockTileDefinitionBuilder, ThreeDimensionalSceneTile, ThreeDimensionalSceneTileDefinition, Frustum} from "@glowbuzzer/controls";
import {DockExampleLayout} from "@glowsite";

const CustomTile=DockTileDefinitionBuilder(ThreeDimensionalSceneTileDefinition)
    .render(() => {
        return <ThreeDimensionalSceneTile>
            <Frustum scale={1000}/>
            {["red", "green", "blue"].map((colour, index) => (
                <mesh key={index} position={[500, (index - 1) * 200, 75]}>
                    <boxGeometry args={[150, 150, 150]} />
                    <meshStandardMaterial color={colour} />
                </mesh>
            ))}
        </ThreeDimensionalSceneTile>
    })
    .build();

export default () => <DockExampleLayout tile={CustomTile}/>
```
