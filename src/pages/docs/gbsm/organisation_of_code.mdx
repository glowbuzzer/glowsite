# {props.subtitle}

The GBSM code base is not particularly complicated and it should not be too hard for you to get to grips with it.


We use the TMC-API (<a href="https://github.com/trinamic/TMC-API">https://github.com/trinamic/TMC-API</a>) for the definitions of the registers on the two ICS. These are included in the project in:

`libs/TMC-API`
