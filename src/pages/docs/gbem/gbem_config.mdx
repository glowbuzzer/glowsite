# {props.subtitle}

`gbem_config` is a static library in which is the `gbem_config.h` file.

This is where the primary compile time configuration parameters for GBEM are set.

The file is well commented.

Some options are configurable from with the CMake build system - see `gbem_defs_linux.cmake`.
