
We provide binaries for 64-bit Linux and the Raspberry Pi for evaluation. Other platforms can be supplied on request.

Raspberry PI (arm7 32 bit)
`https://downloads.glowbuzzer.com/gbc-armv7.tar.gz`

Linux (64 bit)
`https://download.glowbuzzer.com/gbc/gbc-amd64.tar.gz`

Move to the directory in which you want to install GBC.

Next, download GBC from: `https://download.glowbuzzer.com/gbc-armv7.tar.xz`.

If `wget` needs installing, install with:
```shell
sudo apt install wget
```
Otherwise, just run:

```shell
wget https://download.glowbuzzer.com/gbc-armv7.tar.gz
```

Then uncompresss with:
```shell
tar -xzf gbc-armv7.tar.gz
```
