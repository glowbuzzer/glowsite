# Image using glowsite imagetools param

By default the Image component assumes documentation page (with left nav) and will optimise for approx 70% page split.

<Image meta={img1} alt="Glowsite fancy image"/>

## Go wide

Specify 'wide' preset and Image component will optimise for the full viewport width.

<Image meta={img2} alt="Glowsite fancy image" preset="wide"/>

## Specify a max width (in pixels)

Specify an explicit max-width to prevent image from expanding to full width. Image will still scale down when less
width than specified is available.

<Image meta={img2} alt="Glowsite fancy image" maxWidth={400}/>

## Use narrow preset for multiple columns

Using preset `narrow` will cause a smaller image to be loaded even on wider screens.

<Row>
    <Col span={12}>
        <h2>Col 1</h2>
        <Image meta={img3} alt="Glowsite fancy image" preset="narrow"/>
    </Col>
    <Col span={12}>
        <h2>Col 2</h2>
        <Image meta={img3} alt="Glowsite fancy image" preset="narrow"/>
    </Col>
</Row>
