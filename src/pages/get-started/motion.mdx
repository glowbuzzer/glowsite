# How do I get a drive moving?
<Tldr>
    - Use the getting started hardware to get a drive moving
    - Commission and EtherCAT master
    - Use the web-front-end to jog the drives
    - Interact with the EtherCAT IO
</Tldr>

<Alert message="Caution" description="This guide assumes you are working with the single axis EtherCAT starter kit hardware and it have it fully assembled" type="warning" showIcon style={{margin: '20px 0 20px 0'}} />


## Step 1 - Download the generic front-end application to your PC
````shell
npm i
````
## Step 2 - Download and run GBC on the Raspberry Pi
Connect to the pi with ssh and password `raspberry`
````shell
ssh root@rpi-gb
````
Download GBC from GitHub?
````shell
mkdir glowbuzzer_toolkit
````

## Step 3 - Download GBEM configure it for the test machine hardware
Download from GitHub
````shell
git clone https://gitgub.com/glowbuzzer/GBEM

````

machine config

